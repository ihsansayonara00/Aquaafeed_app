🐠 Aplikasi AquaFeed

Sistem Rekomendasi Pakan Optimal (Java Desktop - Greedy Fractional Knapsack)

Silakan klik tombol di bawah ini untuk mengunduh file aplikasi:

📥 Unduh Aplikasi (.JAR)

📢 Petunjuk Sebelum Menjalankan Aplikasi:

  1. Pastikan komputer Anda sudah terinstal Java Runtime Environment (JRE) 17 atau yang terbaru.
  2. Nyalakan tool database lokal Anda (Laragon atau XAMPP) dan pastikan service MySQL Server dalam keadaan Start / Active.
  3. Import file database Database_apk.sql yang tersedia di dalam folder proyek jika diperlukan.
  4. Klik dua kali (double-click) pada file .jar yang telah diunduh untuk membuka aplikasi.
